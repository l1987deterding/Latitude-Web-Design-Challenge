Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes with proximity to the equator. OpenWeatherMap API was used to generate the dataset used for the analysis.

Matplotlib was used to plot temperature, cloudiness, wind speed, and humidity against latitude. Provided here is the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
